# OTOI Examples

> **Legacy examples:** The JSON examples in this directory currently use older
> TOI and Collaborative Charter shapes. They remain useful for understanding
> accessibility needs and interaction intent, but new machine-readable examples
> should use canonical `.toi` documents plus `.otoi` charters. Use
> [`docs/canonical-toi-migration.md`](/docs/canonical-toi-migration.md) when
> translating these examples.

This directory contains real-world examples of OTOI documents to help you understand how to create your own Terms of Interaction.

## Organization

### [Neurodivergent Examples](/examples/neurodivergent-examples/)
Examples specifically designed by and for neurodivergent individuals:
- ADHD-friendly interaction patterns
- Autism-accommodating communication styles  
- Executive function support preferences
- Sensory consideration examples

### [Team Collaboration](/examples/team-collaboration/)
Examples of Collaborative Charters for group AI interaction:
- Small team protocols
- Large organization frameworks
- Remote work adaptations
- Cross-cultural considerations

### [AI Assistant Configs](/examples/ai-assistant-configs/)
Examples tailored for specific AI assistant use cases:
- Personal productivity assistants
- Learning and education support
- Creative collaboration
- Professional task assistance

## Using These Examples

1. **Browse by Category**: Find examples similar to your situation
2. **Study the Patterns**: Notice how preferences are expressed
3. **Adapt, Don't Copy**: Use these as inspiration for your own TOI
4. **Contribute Your Own**: Share examples that help others

## Example Format

Each example includes:
- **Context**: Who this is for and what situation
- **Key Features**: What makes this example useful
- **Full TOI**: The complete document
- **Adaptations**: How it might be modified for other uses

## Privacy Note

All examples are either:
- Created specifically for educational purposes
- Anonymized versions of real TOIs (with permission)
- Composite examples based on common patterns

No personal information is included without explicit consent.

## Contributing Examples

Have a TOI that works well for you? Consider contributing an anonymized version:

1. **Anonymize**: Remove all personal identifying information
2. **Generalize**: Make it useful for others in similar situations
3. **Document**: Explain what makes it effective
4. **Share**: Submit through our contribution process
